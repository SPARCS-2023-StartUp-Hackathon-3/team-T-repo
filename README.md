## Title: AI 창작품의 생성 및 관리를 도와주고, 창작품을 공유하는 SNS형 커뮤니티 플랫폼

다음과 같은 기능이 포함되어 있습니다.
- Prompt 입력 시 <b>AI 창작품 생성</b>
- <b>장르별</b> 작품 열람
    - 만화풍, 실사풍, 3D 등
    - 각 작품의 주제, 카테고리, 테마별 프롬프트 정리 및 요약본 열람
- <b>AI 작품 업로드</b>에 최적화된 인터페이스 제공
- (로그인 상태 한정) 개인 계정별 SNS 기능
    - AI 작품 생성 및 업로드
    - Google 로그인 연동
